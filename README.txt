Project 1

Execution instructions

chatserve.py:
To execute chatserve.py,
1) Ensure that chatserve.py is located in the current working directory
2) Ensure that chatserve.py is set as executable (e.g., chmod u+x ./chatserve.py)
3) Execute ./chatserve.py [PORTNUM]

chatclient:
To execute chatclient,
1) Ensure that chatclient.c and makefile are located in the current working directory
2) Type make to compile
3) Execute ./chatclient [HOST] [PORTNUM]
